## خطوات الإضافة السريعة

### إذا أردت إضافة خدمة جديدة الآن:

1. **افتح الملف:**
   ```
   src/data/enhancedServices.ts
   ```

2. **انسخ أي خدمة موجودة (seo أو ui-ux)**

3. **غيّر 3 أشياء فقط:**
   - slug: ضع معرف فريد
   - title: ضع عنوان الخدمة
   - summary: وصف قصير

4. **احفظ الملف وجدّد المتصفح (Ctrl+Shift+R)**

5. **جرّب الرابط الجديد:**
   ```
   http://localhost:5173/services/[slug-الجديد]
   ```

## المشاكل الشائعة والحلول

### ❌ "Localhost لا يعمل"
**الحل:** تأكد من:
```bash
npm install  # تثبيت المكتبات
npm run dev  # تشغيل الموقع
```

### ❌ "الصفحة بيضاء"
**الحل:** 
```bash
npm run build  # حاول البناء مرة أخرى
npm run dev    # ثم شغّل الموقع
```

### ❌ "Ctrl+Shift+R لا تحدّث"
**الحل:** افتح DevTools (F12) وانقر يمين على refresh مع اختيار "Empty Cache and Hard Refresh"

### ❌ "النصوص مختلطة (عربي/إنجليزي)"
**الحل:** تأكد من إضافة:
```typescript
{
  ar: 'النص بالعربية',
  en: 'Text in English'
}
```
