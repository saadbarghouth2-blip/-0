
## قائمة التحقق 📋

### التحميل والأداء:
- [ ] الصفحة تحمل بسرعة
- [ ] الأنيميشنات سلسة وليست متقطعة
- [ ] لا توجد مشاكل في وحدة التحكم (Console)

### العناصر التفاعلية:
- [ ] يمكن النقر على التبويبات وتتحول الأقسام
- [ ] الأسئلة الشائعة تتوسع عند النقر
- [ ] الأزرار تعطي تجاوب بصري عند التمرير
- [ ] الانتقالات بين العناصر سلسة

### التصميم والمحتوى:
- [ ] المحتوى واضح وسهل القراءة
- [ ] الألوان متسقة وجذابة
- [ ] الفراغات والتباعد متوازن
- [ ] الخطوط واضحة على الموبايل

### اللغة والترجمة:
- [ ] جميع النصوص بالعربية في النسخة العربية
- [ ] جميع النصوص بالإنجليزية في النسخة الإنجليزية
- [ ] التبديل بين اللغات يعمل بسلاسة

### الموبايل والاستجابة:
- [ ] الصفحة تبدو جيدة على الموبايل
- [ ] العناصر لا تتداخل على الشاشات الصغيرة
- [ ] يمكن التمرير بسهولة
- [ ] الأزرار بحجم قابل للنقر

```

```

### 5. CTA Section
```
✓ الزر يرسل للصفحة المناسبة
✓ النص واضح والألوان جذابة
✓ التأثير عند التمرير يعمل
```

### 6. Related Services
```
✓ تظهر خدمات أخرى ذات صلة
✓ يمكن النقر عليها للانتقال
✓ الروابط تعمل بشكل صحيح
```

## اختبار الأخطاء الشائعة

### إذا كانت الصفحة فارغة:
```bash
# تحقق من:
1. هل الخدمة موجودة في enhancedServices.ts؟
2. هل المسار صحيح؟
3. افتح وحدة التحكم (F12) للأخطاء
```

### إذا كانت الأنيميشنات بطيئة:
```bash
# جرب:
1. أغلق التبويبات الأخرى
2. فعّل hardware acceleration في المتصفح
3. تحقق من الأداء مع DevTools
```

### إذا كانت النصوص مختلطة:
```bash
# تحقق من:
1. قيمة lang في useLanguage hook
2. هل البيانات تحتوي على قيم ar و en؟
3. افتح وحدة التحكم للتحقق من الأخطاء
```

## اختبار الأداء

### استخدام DevTools:
1. افتح F12
2. اذهب إلى Performance tab
3. اضغط Record
4. تفاعل مع الصفحة
5. اضغط Stop
6. تحليل النتائج

### مقاييس الأداء المثالية:
- **FCP** (First Contentful Paint): < 1.8s
- **LCP** (Largest Contentful Paint): < 2.5s
- **CLS** (Cumulative Layout Shift): < 0.1
- **TTI** (Time to Interactive): < 3.5s

## الإجراءات عند العثور على مشكلة

### 1. وثّق المشكلة:
- [ ] ما هي الخطوات لإعادة الإنتاج؟
- [ ] متى يحدث؟
- [ ] على أي متصفح/جهاز؟

### 2. افتح وحدة التحكم:
- [ ] هل هناك أخطاء في Console؟
- [ ] هل هناك تحذيرات في Network tab؟

### 3. تحقق من الكود:
- [ ] هل الخطأ في المكون أم في البيانات؟
- [ ] هل يمكن استنساخ الخطأ بسهولة؟
